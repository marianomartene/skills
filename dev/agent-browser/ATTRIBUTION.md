## Notes

`SKILL.md` is a discovery stub. The actual workflow content is served by the
CLI, which must be installed separately:

```bash
npm i -g agent-browser && agent-browser install
```

Once installed, load the live workflow guide with `agent-browser skills get core`.
No changes have been made to the upstream `SKILL.md`.
